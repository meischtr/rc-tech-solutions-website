# RC-Tech Solutions — Company Website

**Firmenwebsite von RC-Tech Solutions (rc-tech.ch)**

Dies ist die Unternehmenswebsite der Firma RC-Tech Solutions.
Nicht zu verwechseln mit der Saasiro-Produktwebsite.

## Projektstruktur im Ueberblick

```
claude-projekte/
├── rc-tech-solutions-company-website/  <-- DU BIST HIER (Firmenwebsite rc-tech.ch)
├── rc-tech-solutions-website/          <-- Saasiro Produktwebsite (saasiro.com)
├── rc-tech-solutions-gmbh/             <-- Firmen-Dokumente (Vertraege, Compliance, Marketing)
├── neoverse-app-center/                <-- Saasiro Platform Backend (ehem. Neoverse AppCenter)
└── neoverse-saas-template/             <-- SaaS Template fuer Saasiro-Kunden
```

## Technologie

Standalone HTML/CSS/JS — kann spaeter zu Astro migriert werden.
